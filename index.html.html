HTML Intraduction

HTML

HTML is a markup language used to define the structure of web documents which includes text, images, links, forms, and other content.

Basic Structure of an HTML

1.!DOCTYPE html: Declares the document type and version (HTML5).
2.html:Root element that encapsulates all content.
3.head: Contains meta-information about the webpage, such as title, character set, and links to stylesheets.
4.body: Contains the content of the webpage, such as headings, paragraphs, and links.

Types of Tages

In HTML, tags are used to define elements on a webpage. These tags are categorized into paired tags and unpaired tags also called self-closing tags.


1. Paired Tags (Container Tags):paried tages consist of both opening and closing tages
EX:h1,p,head,b....
2. Unpaired Tags (Self-Closing Tags):Unpaired tags, also known as self-closing tags, do not have a separate closing tag.
EX:img,br,hr,input......

Elements of html

 The combination of opening and closing tags and with content between the tags is konw as elememts.elements can have attributes also.

Types of elements in html

 1. Block level element:The tag which occupies whole width and creates a new line
EX: h1-h6,P, div,semantic elements
2.Inline block level element :The elements which occupy only require space and does not create a new line and we can apply width,height,margin andpadding properties.
EX:img, iframe,input, button.
3.Inline element:The elements which occupies only required space and does not create a new line,in css we cannot apply width, height,padding top and bottom and margin top and bottom.
EX:sapn, formatting tags, b, strong,.......
